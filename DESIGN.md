# 珩小派科技有限公司官网 DESIGN.md

## 1. 品牌核心

珩小派科技有限公司是一张面向外界的数字名片，也是一家围绕 AI、人工智能、一人公司 OPC 的高杠杆科技工作室。

品牌关键词：

- 精密 Precision
- 克制 Restraint
- 可信 Trust
- 高杠杆 Leverage
- 实验性 Experimental
- 长期主义 Long-termism

品牌口号建议：

> 小体量，不小气场。以 AI 放大个体创造力，以系统化方法交付长期价值。

## 2. 设计原则

### 不做什么

- 不使用机器人头像、脑机图标、廉价节点连线。
- 不大面积堆叠紫蓝 AI 渐变。
- 不用满屏漂浮卡片制造“AI 模板味”。
- 不为了炫技牺牲加载速度、可读性和可访问性。

### 要做什么

- 深色黑场作为底，使用少量冷暖光源建立层次。
- 3D 粒子和光影只做“能力氛围”，不抢走文案主导权。
- 中文排版优先，控制行高、字距、段落呼吸感。
- 动效遵循“进入、停顿、响应、离开”的节奏，不持续躁动。

## 3. 色彩系统

| Token | Value | 用途 |
| --- | --- | --- |
| `bg.canvas` | `#0A0D12` | 页面底色 |
| `bg.surface` | `#10151C` | 卡片、导航、浮层 |
| `line.hairline` | `rgba(255,255,255,0.10)` | 分隔线、边框 |
| `text.primary` | `#F5F7FA` | 主文字 |
| `text.secondary` | `#AAB3C2` | 辅助文字 |
| `accent.cool` | `#5D88FF` | 科技冷光 |
| `accent.warm` | `#C6A06A` | 温润金属色 |
| `accent.signal` | `#8EF0C7` | 状态/成功反馈 |

色彩面积原则：强调色不超过页面面积 12%。高级感来自节制，不来自堆满渐变。

## 4. 字体与排版

- 中文主字体：Noto Sans SC / Source Han Sans CN / 系统黑体
- 英文与数字：Inter / system-ui
- 正文行高：1.65–1.8
- 标题行高：1.1–1.25
- 大标题优先使用字重和留白塑造气场，不依赖奇形怪状字体。

## 5. Logo 方向

推荐主方向：**珩轴 Heng Axis**

- 以“珩”的平衡、横向轴线、精密结构为核心。
- 结合 H 的几何骨架，形成稳定、可信、可扩展的品牌标识。
- 小尺寸图标使用“引力核”：一个核心点 + 一段轨道弧线。

Logo 语义：

- 珩：平衡、秩序、精密、玉石质感。
- 小派：小体量、高效率、派生、分发、智能编排。

## 6. Icon 系统

- 24px 栅格
- 2px 主笔画
- 圆角矩形端点
- 单一拐角半径体系
- 适度引入“点、线、轨道、切角”作为专属语言

## 7. UI 组件风格

- 卡片：深色半透明表面 + 1px hairline + 轻微内发光。
- 按钮：磁吸 hover、光晕扫过、压下轻微位移。
- 导航：低存在感、滚动后轻微毛玻璃。
- 表单：清晰 label、可访问 focus、提交态明确。
- 数据卡：不要做廉价仪表盘，以编辑式信息密度呈现。

## 8. 动效原则

动效关键词：quiet, cinematic, precise, magnetic, orbital。

- 页面进入：轻微 opacity + y 轴位移。
- Hover：磁吸、边缘光、细微角度变化。
- 滚动：使用分段叙事，不做长时间强制滚动劫持。
- 3D：跟随鼠标产生视差和粒子扰动。
- Reduced Motion：必须提供静态降级。

## 9. 3D / 粒子视觉

推荐视觉组合：

- 点云 Particle Field
- 轨道线 Orbital Path
- 低强度 Bloom
- 细颗粒 Noise
- 少量金属暖光

避免：

- 过度炫光
- 粒子太多导致廉价屏保感
- 每个模块都上 3D

## 10. 首页叙事结构

1. Hero：建立气场与定位。
2. What：说明珩小派做什么。
3. Method：展示 AI / OPC / 自动化编排方法论。
4. Capabilities：产品、咨询、自动化、内容与增长系统。
5. Proof：项目记录、博客、持续学习与公开构建。
6. Contact：公众号、博客、邮箱或后续表单。

## 11. 工程验收标准

- Lighthouse 性能目标：移动端尽量 80+，桌面 90+。
- 3D 组件必须懒加载。
- 首屏主文案不可依赖 Canvas 渲染。
- 支持 `prefers-reduced-motion`。
- 首页无 3D 时也必须足够高级。
- 代码结构清晰，后续可扩展博客、案例、产品页。
